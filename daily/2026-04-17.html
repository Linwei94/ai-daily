Seedance 2.0: Advancing Video Generation for World Complexity

ByteDance Seed字节跳动旗舰视觉团队 · Seedream / Seed-VL 同班底

下一代视频生成模型,首次将文本、图像、音频、视频统一进单一多模态扩散管线,单次推理产出带原生双耳音频、多镜头切换、真实物理交互的长视频(可达 90 秒)。

关键技术: 基于 Seed-VL 视觉语言主干 + Diffusion Transformer 主架构;引入 SeedVideoBench 2.0 评测集(含 motion quality / audio-visual sync / narrative quality / cinematographic language 四维度);训练管线整合 T2V / I2V / R2V 三模态。
核心突破: 不再用单独的视频→音频后处理,而是在生成阶段联合建模音视频时序一致性,这是过去 Sora、Veo 都没解决的问题。
为何重要:Seedance 2.0 的公测 API 04-14 上线 fal,商用就绪。视频生成正式进入"导演级控制 + 长片生成"阶段,创作工具链可能在 Q3 重新洗牌。

ClawGUI: A Unified Framework for Training, Evaluating, and Deploying GUI Agents

浙江大学 ZJU-REALFei Tang*, Zhiqiong Lu*, Boxuan Zhang, Weiming Lu, Jun Xiao, Yueting Zhuang, Yongliang Shen†

把 GUI Agent 全流程(训练 / 评测 / 部署)收进单一框架。包含三大模块:ClawGUI Framework(基础设施)、ClawGUI-RL(强化学习训练栈)、ClawGUI-Eval(基准评测)、ClawGUI-Agent(可部署的 2B/8B 模型)。

关键技术: 训练栈用 GiGPO 算法 + Process Reward Model + Docker Android Emulator 并行环境;MLLM-as-Judge 做自动评分;支持 Hybrid CLI-GUI 控制。
实验亮点: 在 AndroidControl / GUI-G2 / MMBench-GUI 多个 benchmark 上,2B 小模型即可对标 GUI-Owl / Aguvis 等专门化 Agent。
为何重要:GUI Agent 是当下最热的 Agent 落地方向(电商风控 / 办公自动化 / 移动端任务)。各家方案过去碎片化,ClawGUI 一统训练-评测-部署,有望成为参考实现。

WildDet3D: Scaling Promptable 3D Detection in the Wild

UW PRIOR × Cornell × AI2Weikai Huang, Jieyu Zhang, Jaemin Cho · Ali Farhadi, Ranjay Krishna (UW PRIOR) · Bharath Hariharan (Cornell) · 17 作者

把 promptable 3D 检测从封闭 benchmark 推向真实野外开放场景。沿着 SAM 在 2D 的"prompt + 大数据 + 大模型"路径,用 scaling 来攻克 3D 长尾问题。

关键技术: 大规模野外 LiDAR + 多视角图像数据集;promptable 检测头;text/box/point 多种 prompt 模态。
核心贡献: 证明 3D 检测同样存在 scaling law,模型 + 数据规模上来后,zero-shot 泛化到未见场景的能力跃升。
为何重要:自动驾驶 / 机器人感知的 3D 检测过去严重依赖闭域微调。如果开放世界 3D foundation model 能成立,Tesla / Waymo 之外的玩家也能用通用模型起步。

The Past Is Not Past: Memory-Enhanced Dynamic Reward Shaping

复旦大学 FudanNLPYang Liu, Enxi Wang, Yufei Gao, Weixin Zhang, Bo Wang, Zhiyuan Zeng, Yikai Zhang, Yining Zheng, Xipeng Qiu(邱锡鹏)

在 RL 训练中引入显式记忆模块,让 agent 能利用历史 trajectory 动态调整 reward 信号 —— 解决长 horizon、sparse reward 任务里"近期成功被遗忘 / 重要事件被淹没"的问题。

关键技术: Memory-augmented value function;基于历史经验的 dynamic reward shaping;可与 PPO / GRPO 等主流 RL 算法插拔组合。
实验亮点: 在 sparse reward 控制任务(机器人导航、工具使用)上,样本效率提升 30-50%,尤其在 episode 长度 >200 时优势显著。
为何重要:LLM Agent 的 RL 后训练正卡在 long-horizon credit assignment 这一关。这篇可能直接迁移到 LLM agent training 里,改善多轮工具调用 / 多步推理的训练稳定性。

RationalRewards: Reasoning Rewards Scale Visual Generation Both Training and Test Time

HKUST × Waterloo × AlibabaHaozhe Wang (HKUST), Cong Wei / Weiming Ren / Wenhu Chen (Waterloo), Jiaming Liu (Alibaba), Fangzhen Lin (HKUST)

把 LLM 里成功的 reasoning reward 范式迁移到视觉生成 —— 训练阶段用推理型 reward 指导 diffusion 优化(DiffusionNFT),测试阶段用 Generate-Critique-Refine 循环精修。

关键技术: PARROT 三阶段管线(hindsight 标注 / consistency filter / foresight SFT);Process Reward Model 基于 Qwen3-VL-32B;支持 LoRA 轻量化部署;明确防止 reward hacking 设计。
实验亮点: 在 GenAI-Bench / GEdit-Bench-EN / HPDv3 多个偏好评测上,8B reward model 同时改善训练阶段(+15% 偏好胜率)和测试阶段(+8% 一次生成质量)。
为何重要:LLM 的 reasoning reward 是 o1/o3 路线的核心,本工作首次系统迁移到视觉生成。可能是下一波 SOTA 视觉模型的统一 recipe。
