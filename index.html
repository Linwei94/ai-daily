RAD-2: Scaling Reinforcement Learning in a Generator-Discriminator Framework

华中科技大学 HUSTRAD 系列第二代;沿用 GAN 思路做 RL scaling

把 RL 重新表述为 generator-discriminator 博弈,让 reward 信号天然 dense 且可 scale。在大模型 RL 后训练阶段(类 PPO/GRPO 替换),探索新的 scaling law。

关键技术: Generator(策略)输出 trajectory,Discriminator 区分专家与策略输出 → discriminator 概率即 reward。规避 PPO 的 sparse reward 与 reward hacking。
实验亮点: 在 LLM math / code RL 上较 GRPO baseline 提升明显,且 scaling 曲线更平滑。
为何重要:RL 后训练是当前 LLM 提升 reasoning 的核心瓶颈;PPO/GRPO 卡在 sparse reward。GAN 式 dense reward 是新方向之一,值得跟踪。

DR³-Eval: Towards Realistic and Reproducible Deep Research Evaluation

19 作者Deep Research Agent 专门评测集

现役 Deep Research Agent(OpenAI / Perplexity / Gemini DR)缺乏可复现评测。DR³-Eval 提出"真实 + 可复现"双轴 benchmark,固化网页快照、引用核验、报告完整度三档评分。

意义: Deep Research 是 2025-2026 最热的 agent 落地方向之一,但评测一直靠各家自说自话。DR³ 可能成为参考标准。
为何重要:你写 ai-daily 简报的工作流本质上就是个 Deep Research Agent。这篇能直接用来评测 / 改进 /ai-daily 的质量。

GlobalSplat: Efficient Feed-Forward 3D Gaussian Splatting via Global Scene Tokens

耶路撒冷希伯来大学 HUJI3DGS feed-forward 方向新作

前馈式 3D Gaussian Splatting(从 sparse 视图直接预测 3D 高斯,不要 per-scene 优化)的效率痛点是 token 太多。GlobalSplat 用 global scene tokens 压缩表征,在保质前提下显著加速。

关键技术: 引入"global tokens"作为场景级 anchor,所有 patch tokens 通过 cross-attention 与之交互 → token 数量从 O(N) 降到 O(K),K 远小于 N。
定位: Feed-forward 3DGS 方向(LRM、Splatter Image、pixelSplat、MVSplat 系列)的最新效率优化。
为何重要:3DGS 是 NeRF 之后的事实标准,feed-forward 方向决定了能否做"实时端到端 3D 生成"。这条线上的进展,世界模型 / AR / 机器人都吃。
